# 📱 Mobile Application Development (MAD) Lab

A curated collection of Mobile Application Development (MAD) experiments, assignments, and projects completed during the **2026–2027 academic year**.

This repository serves as a structured record of practical work, covering Android application development concepts, user interface design, activity lifecycle, data storage, API integration, and modern mobile development practices. It is intended for learning, experimentation, and academic reference.

---

## 📌 Objectives
* Build a strong foundation in Android application development.
* Design intuitive and responsive mobile user interfaces.
* Implement core Android components and application architecture.
* Explore local data storage, networking, and API integration.
* Develop functional mobile applications following modern development practices.
* Maintain a well-organized archive of laboratory experiments and coursework.
